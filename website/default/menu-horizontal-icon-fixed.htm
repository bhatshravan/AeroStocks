Horizontal Fixed with icon layout is useful for those users who only wants menu options on top.
CSS Class
To use static layout for your project add .horizontal-icon class in body tag.
JS Option
To use Compact Menu for your project add themelayout: 'horizontal', FixedNavbarPosition: true, FixedHeaderPosition: true, class in js.
HTML Markup

Use the following code to use Horizontal Fixed Layout with Icon in horizontal.

Usage:
                                                            
                                                                $( document ).ready(function() {
                                                                    $( "#pcoded" ).pcodedmenu({
                                                                        themelayout: 'horizontal',
                                                                        FixedNavbarPosition: true,
                                                                        FixedHeaderPosition: true,
                                                                    });
                                                                });

                                                                <!DOCTYPE html>
                                                                <html lang="en">
